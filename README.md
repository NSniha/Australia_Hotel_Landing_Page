# Australia Hotel — Premium Hotel Booking Website

Australia Hotel is a modern, responsive hotel booking website designed for premium Australian stays. The project includes a polished homepage, curated hotel room sections, guest testimonials, an About page, a Travel Guide blog, a Guest Support contact page, and an interactive Legal Center.

The website is built with **HTML5, CSS3, and vanilla JavaScript**. It focuses on clean UI design, premium typography, responsive layouts, SEO-friendly structure, and smooth front-end interactions.

---

## Project Overview

Australia Hotel is a multi-page front-end website created for a hotel, travel, or accommodation brand. The goal of this project is to provide users with a refined browsing experience where they can explore rooms, read hotel-related content, contact guest support, and review important policies.

The design follows a premium black-and-white visual system with soft shadows, rounded cards, elegant spacing, and strong visual hierarchy. Every page is structured to feel consistent, professional, and easy to use across desktop, tablet, and mobile devices.
